# Rise at Seven - MERN Stack Project

A modern web application built with MERN stack (MongoDB, Express, React, Node.js) showcasing the Rise at Seven agency portfolio and services.

## Features

- **Responsive Design**: Fully responsive interface that works on desktop, tablet, and mobile devices
- **Smooth Animations**: Glassmorphism effects, scroll animations, and smooth transitions
- **Modern UI**: Contemporary design with gradient backgrounds and interactive components
- **Navbar Behavior**: Smart navbar that hides when certain sections are fully visible
- **Featured Work Section**: Showcase of client projects with scroll-based animations
- **Legacy In The Making**: Three-card carousel with scroll animations
- **Services Section**: Comprehensive service offerings with marquee animation

## Tech Stack

- **Frontend**: React 18 with Framer Motion for animations
- **Styling**: Tailwind CSS with custom configurations
- **Build Tool**: Vite for fast development and optimized builds
- **Motion Library**: Framer Motion for complex animations

## Project Structure

src/

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Fuad-Rafi/Rise-at-seven.git
cd Rise-at-seven
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally

## Components

### Header
Fixed navigation bar with glassmorphism effect. Automatically hides when Hero, FeaturedWork, or LegacyInTheMaking sections are fully visible.

### Hero Section
Large hero section with animated text and platform badges.

### Featured Work
Horizontal scroll-based showcase of client projects with smooth animations.

### Services Section
Grid layout of service offerings with marquee animation banner.

### Legacy In The Making
Three-card carousel with rotation and scroll-based animations.

### WhatsNew
Blog-style news and updates section.
